Features:

  1. Adj. closing prices from 500 stocks
  2. Uses NASDAQ composite as our index, with data from 2004-2018
  3. Python library for downloading/querying data custom written
  4. Error handling on failed downloads and on querying for missing stocks
  5. Plotting supports panning, zooming, muting/hiding stocks, etc (via Bokeh library)
  6. Betas calculated on the fly based on cached pricing information
